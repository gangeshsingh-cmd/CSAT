# 7-Day CSAT Booster Plan

A comprehensive study guide for UPSC Prelims Paper II (CSAT) designed to build confidence and mastery in just 7 days.

## 📚 Course Structure

Each day covers one core chapter with:
- **Concept Explanation**: Clear definitions and rules
- **CSAT Shortcuts**: Exam-specific tips for quick solutions
- **Solved Example**: Step-by-step walkthrough
- **3 Practice Questions**: Exam-level problems with answers

## 📅 Daily Topics

1. **Day 1**: Number System
2. **Day 2**: Divisibility & Remainder
3. **Day 3**: Percentage
4. **Day 4**: Average
5. **Day 5**: Speed, Time & Distance
6. **Day 6**: Syllogism
7. **Day 7**: Direction & Distance

## 🚀 Getting Started

1. Open `index.html` in your browser
2. Follow the daily tracker
3. Complete each day's concept, example, and practice questions
4. Check answers and review mistakes

## 📊 What You'll Get

- ✅ 7 Core Chapters
- ✅ 7 Solved Examples
- ✅ 21 Practice Questions
- ✅ 33% Qualifying Target Framework
- ✅ Printable PDF Format

## 💡 Study Approach

**Each day takes ~30-40 minutes:**
1. **Samajho (10 min)**: Study concept & shortcuts
2. **Dekho (10 min)**: Solve example yourself first
3. **Karo (15 min)**: Complete 3 practice questions
